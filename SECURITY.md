# Security Policy

## Supported Versions

Security updates are provided for the latest maintained version of **JP2 Air Quality Card**.

If you are using an older version, please update to the most recent release before reporting a security issue.

## Reporting a Vulnerability

If you discover a security issue, please do **not** open a public GitHub issue.

Instead, report it privately with the following details:

- A clear description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Any logs, screenshots, or proof of concept that may help
- The version of the card you are using

Please provide enough information to allow the issue to be reproduced and verified.

## What to Expect

After a report is received:

- The issue will be reviewed as quickly as possible
- A fix will be prepared if the report is confirmed
- A security release may be published when required
- Responsible disclosure is appreciated until a fix is available

## Scope

This policy applies to:

- The card source code
- The generated distribution file
- Editor-related configuration behavior
- Any code included in this repository

It does not apply to:

- Third-party integrations
- Home Assistant core itself
- Custom configurations outside this repository

## Responsible Disclosure

Please avoid publicly sharing exploit details before the issue has been reviewed and fixed.

This helps protect all users of the project.

Thank you for helping keep **JP2 Air Quality Card** secure.
